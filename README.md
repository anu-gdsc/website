# GDSC ANU Website

Official website for the Google Developer Student Club (GDSC) at the Australian National University (ANU), built with a focus on performance, scalability, and modern UI/UX.

---

## 🚀 Project Overview

This project is being developed using **Next.js (App Router)** and **Tailwind CSS** to create a fast, responsive, and visually engaging experience for students and developers.

The goal is to build a **central platform** for:

* Events and workshops
* Speaker showcases
* Community engagement
* Resources and updates

---

## 📌 Current Status

🚧 **UI Development Phase**

* The application currently uses **static / placeholder data**
* Focus is on **component architecture, responsiveness, and design system**
* Codebase is structured to support **scalable future integrations**

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/gdsc-anu/<repo-name>.git
cd <repo-name>
```

---

### 2. Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

---

### 3. Run the development server

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

---

## 🏗️ Project Structure

```
/app        → Routing & pages (Next.js App Router)
/components → Reusable UI components
/lib        → Utilities & helpers
/hooks      → Custom React hooks
/styles     → Global styles
/public     → Static assets
```

## 🤝 Contributing

We follow a **PR-based workflow** to maintain code quality.

### Steps:

1. Fork or clone the repo
2. Create a new branch

   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Make your changes
4. Commit your work

   ```bash
   git commit -m "feat: add your feature"
   ```
5. Push to your branch
6. Open a Pull Request
